BELOW ARE A FEW HELPFUL RECURRING PATTERNS I’VE ENCOUNTERED WHILE SOLVING MONOTONIC TYPE PROBLEMS

-> While solving the problem we encounter a scenario where we have a subarray and a subproblem requires us to consistently perform an 
   operation on the least or highest value in the subarray, usually multiple times to maintain an order 
 ex:- SubarrayWithElementsGreaterThanVaryingThreshold.py

-> Most evident clue, we need to maintain an order of highest or lowest values in order 
 ex:- MaximuminASlidingWindow.js; we need to find maximum in a window but finding max is a O(n) operation, 
      maintaining monotonic order of decreasing elements makes it a O(1) operation

   based on this relationship
